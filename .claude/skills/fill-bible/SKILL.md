# 填充设定

引导用户逐步填充 `bible/` 下的模板文件，完成世界观搭建。

## 流程

按以下顺序，每次只问一个方面，等用户回答后再填充对应文件：

### 1. 故事背景
- 问：时代（现代/民国/近未来/...）、城市特征、社会氛围
- 填充：`bible/universe/setting.md`

### 2. 主角设定
- 问：姓名、年龄、职业、性格核心特质、推理风格、标志性习惯、口头禅
- 请用户给 3 句示例台词
- 填充：`bible/characters/protagonist.md`

### 3. 搭档设定
- 问：姓名、与主角关系、性格（需与主角互补）、说话方式
- 填充：`bible/characters/sidekick.md`

### 4. 常驻配角
- 问：有没有想好的配角？（可以跳过，后续再补）
- 填充：`bible/characters/recurring.md`

### 5. 世界观规则
- 展示 `bible/universe/rules.md` 当前内容，问用户是否需要调整
- 重点确认：技术水平、推理规则、暴力尺度

### 6. 暗线大纲
- 问：大阴谋是什么？幕后黑手是谁？与主角什么关系？
- 引导用户设计三阶段暗线（暗流→浮现→对决）
- 填充：`bible/arc/master-arc.md`

### 7. 案件池
- 问：有没有想好的案件创意？
- 如果有：帮用户整理成标准格式填入
- 如果没有：根据已有设定自动生成 10 个案件梗概
- 填充：`bible/arc/case-pool.md`

## 完成后

展示设定摘要，确认无误后告知用户：

```
设定完成！现在可以使用 /write-chapter 开始写第一章。
```
