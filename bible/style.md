# 写作风格指南

本指南是 AI 写作 Agent 的操作手册，适用于系列悬疑推理小说的章节生成。所有规则均为强制执行项，不得以"文学自由"为由绕过。

---

## 一、基本参数

| 项目 | 规格 |
|------|------|
| 视角 | 第三人称限知（跟随主角，不得进入其他角色的内心） |
| 时态 | 过去时（"他走进房间"，不用"他走进房间。现在他看到……"） |
| 每章字数 | 3000–4000 字 |
| 章节结构 | 三段式：上篇 / 中篇 / 下篇，各 1000–1300 字 |

**视角纪律**：叙述者只能写主角能观察到、听到、感受到的信息。主角不在现场的事件，只能通过对话、文件、第二手叙述引入。违反此规则属于"角色知识违规"，是硬性错误。

---

## 二、章节格式

每章必须严格使用以下 Markdown 格式，不得随意增删层级：

```
# 第 N 章：{章节标题}

## 上 · {小标题}

{正文，1000–1300 字}

## 中 · {小标题}

{正文，1000–1300 字}

## 下 · {小标题}

{正文，1000–1300 字}
```

- `N` 为章节序号（阿拉伯数字）
- 章节标题简短有力，4–8 字，可以是地点、物件、动作或悬念短语
- 小标题点出该段核心动作或情绪转折，避免剧透下一段

**示例**：
```
# 第 3 章：第二把锁

## 上 · 陌生的气味

## 中 · 她撒了谎

## 下 · 钥匙不见了
```

## 三、三幕节奏规范

### 上篇（开场，快节奏钩子）

- **前 3 句必须建立悬念**：用动作、异常细节或反常对话开场，禁止用背景介绍或天气描写起笔
- 引入章节核心冲突或谜题
- 节奏偏快：短句为主，段落不超过 4 行
- 允许截断感：读者可以不完全理解发生了什么，但必须想继续读

**合格开场示例**：
> 门缝里塞着一张字条，上面只有四个字：别去地下室。

**不合格开场示例**：
> 这座城市的十一月向来阴冷，街道上行人稀少，李明裹紧了大衣……

### 中篇（发展，交替推进）

- **对话与内心推理交替出现**，比例大致各半
- 每段对话之后，主角必须有一次内心评估（短则一句，长则三句）
- 植入至少一个**误导线索**：看起来重要、实则无关，或看起来无关、实则关键
- 误导不得靠信息隐瞒实现，只能靠主角的错误推断或不完整观察
- 中篇结尾必须留下一个未解决的小疑问，为下篇铺垫

### 下篇（高潮，加速收束）

- 节奏持续加速：段落越来越短，对话越来越急促
- 揭示一个关键信息（可以是线索确认、人物动机暴露、事件反转）
- **章节结尾禁止出现道德总结、感悟句、人生哲理**
- 结尾必须是以下三类之一：
  1. 行动截断（主角正在做某事，章节戛然而止）
  2. 新谜题抛出（一个新的异常出现）
  3. 情绪余震（一句话，留白，让读者自行填充含义）

## 四、语言风格

### 总原则

- **简洁优先**：能用 10 字说清楚的，不用 20 字
- **禁止形容词堆叠**：每个名词最多跟一个修饰语。"黑暗潮湿破旧的地下室"改为"地下室，墙壁渗着水"
- **具体胜过抽象**：不写"她很紧张"，写"她把杯子放下去，又立刻端起来"

### 感官细节（案发现场专项）

案发现场必须覆盖至少 3 种感官，且细节须服务于推理，不得仅为气氛装饰：

- **视觉**：光线方向、阴影、颜色异常（血迹的颜色会说明时间）
- **嗅觉**：最容易被遗忘、最难伪造，优先使用（烟味、消毒水、体味、食物）
- **听觉**：环境音的缺席有时比存在更重要（"房间里没有钟声"）
- **触觉**：温度、材质（地板是否有水迹、把手是否温热）
- 味觉：仅在主角直接接触时使用，不强求

**错误示范**：
> 现场一片狼藉，令人不寒而栗。

**正确示范**：
> 茶杯还温着。窗帘拉到一半，光线斜切进来，照出地板上一道浅浅的划痕，方向朝向书架。

### 逻辑推理的嵌入方式

推理过程不得以独立的"分析段落"形式出现，必须嵌入动作或对话中：

- **动作驱动推理**：主角拿起某个物件时，顺带想到它的含义
- **对话触发推理**：对方说了某句话，主角在心里标记为"这句话有问题"
- **推理结论延迟**：主角注意到线索，但不立刻说出结论——结论在下篇或下一章才浮现

禁止写法：
> 李明开始分析现场。首先，凶器是……其次，嫌疑人有三个……最后，他得出结论……

## 五、对话规范

### 说话标签

- 优先使用"说"和"问"，其次"答""道"
- 禁止使用情绪化标签替代实际写作：不用"他愤怒地喊道""她悲伤地低语"——情绪由台词内容和前后动作体现
- 标签之后禁止再加副词："他轻声地说"改为"他放低了声音"（动作前置）

### 人物辨识度

每个主要角色须有固定的语言习惯，通过台词本身即可识别说话者：

| 类型 | 实现方法 |
|------|----------|
| 说话节奏 | 长句 vs 短句；爱用省略号还是句号 |
| 口头禅 | 固定的开场词、结尾词、语气词 |
| 方言痕迹 | 特定词汇选择（"晓得"而非"知道"，"咋"而非"怎么"） |
| 职业习惯 | 医生说"症状"，警察说"情况"，律师说"从法律角度" |
| 信息密度 | 话多的人 vs 只说关键词的人 |

每章首次出现某角色时，须在台词前用一句动作描写强化辨识度。

### 对话功能检查

每段对话必须同时完成以下至少两项功能：
1. 推进情节（透露新信息）
2. 揭示角色性格
3. 制造或缓解紧张
4. 植入线索或误导

对话禁止用于纯粹的寒暄或天气议论，除非寒暄本身是伪装（角色在用寒暄回避真正的问题）。

## 六、节奏控制

### 句子长度与段落密度

| 场景 | 句长 | 段落行数 |
|------|------|----------|
| 追逐/冲突/对峙 | 5–15 字 | 1–2 行 |
| 日常调查/移动 | 15–30 字 | 2–4 行 |
| 回忆/内心分析 | 20–40 字 | 3–5 行 |
| 对话密集段 | 视内容而定 | 每句单独成段 |

### 对话与叙事的比例

全章对话字数占比：**30%–40%**。低于 30% 说明动作描写过满，高于 40% 说明缺乏内心层次。

---

## 七、禁止列表（硬性规则）

以下内容一旦出现，视为写作失败，必须修改：

### 禁用词组（空洞连接词）

- 众所周知
- 不言而喻
- 毋庸置疑
- 不得不说
- 值得注意的是
- 总的来说
- 由此可见
- 话虽如此

### 禁用写法

| 禁止 | 原因 |
|------|------|
| "突然，他灵光一闪" / "她忽然想到" | 懒惰写法，推理必须有铺垫 |
| 章节结尾的道德感悟 | 破坏悬疑类型感 |
| 主角知道自己不该知道的信息 | 角色知识违规 |
| 形容词三叠："漆黑漆黑的夜晚" | 废话 |
| 全知叙述："与此同时，在另一个地方……" | 违反限知视角 |
| 突发的性格转变无铺垫 | 破坏人物一致性 |

### AI 高频词黑名单

以下词语因 AI 生成文本中出现频率过高，须主动回避，找具体替代表达：

- 错综复杂
- 扑朔迷离
- 若隐若现
- 心中一紧
- 心头一沉
- 豁然开朗
- 抽丝剥茧
- 层层递进
- 娓娓道来
- 不禁（"他不禁想到"）
- 不由得（"她不由得皱眉"）

## 八、自我检查清单

每次生成章节后，按以下顺序逐项验证：

- [ ] 上篇前 3 句是否建立了悬念？
- [ ] 全章是否严格保持第三人称限知？主角有无接触到视角外的信息？
- [ ] 对话比例是否在 30%–40% 之间？
- [ ] 中篇是否有至少一个误导线索？
- [ ] 案发现场是否覆盖 3 种以上感官细节？
- [ ] 推理过程是否嵌入动作/对话，而非单独列出？
- [ ] 下篇是否加速收束，结尾是行动截断/新谜题/情绪余震三类之一？
- [ ] 章节结尾是否没有道德总结？
- [ ] 禁用词组和 AI 高频词是否已全部清除？
- [ ] 每个出场角色的语言习惯是否与角色档案一致？
- [ ] 字数是否在 3000–4000 字范围内？

## 九、参考节奏范例

以下为中篇段落的节奏示范，展示对话与内心推理交替的正确写法：

> "你昨晚几点回来的？"陈督察把笔记本放在桌上，没有打开。
>
> "十点多吧。"宋薇端起茶杯，又放下了，"具体几点我没看表。"
>
> *十点多*。李明在心里记下这个词。案发时间是十点到十一点之间。她给自己留了一个小时的模糊地带。
>
> "你一个人回来的？"陈督察继续问。
>
> "是。"
>
> 只有一个字。李明注意到宋薇的手放在膝盖上，手指微微收紧，但没有抖。不是紧张，是控制。

这段 120 字的片段完成了：推进情节（时间线核实）、揭示角色性格（宋薇的克制，陈督察的技巧）、植入线索（时间窗口的模糊）、主角内心推理（通过斜体嵌入，不打断叙事流）。
