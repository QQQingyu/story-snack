# 世界规则模板

> 说明：本文件规定本系列小说的硬性约束和叙事原则。所有章节必须遵守这些规则，保证读者体验的公平性和一致性。

---

## 一、推理规则（硬性约束）

### 1.1 零超自然原则

- **规则**：所有谜题的答案必须有现实世界的逻辑解释，不得使用超自然、玄学或无法证伪的因素。
- **允许**：罕见的医学现象、特殊化学反应、不常见但真实存在的心理状态。
- **禁止**：鬼魂、超感官知觉、无法解释的"直觉神准"（主角直觉必须事后有逻辑支撑）。
- **灰色地带处理**：若剧情需要涉及迷信/民俗，必须在揭秘时给出科学/社会学解释。

### 1.2 信息渠道合理性原则

- **规则**：主角获取的每条关键信息，必须有交代来源，且来源在情节中须是可信的。
- **禁止**："主角莫名知道"、信息从天而降、证人主动找上门而无任何动机。
- **操作指引**：
  - 情报来源需在获取信息前1-3章内建立（介绍线人、建立关系）。
  - 若使用技术手段（监控、数据库），需说明主角如何合法或非法取得权限。
  - 偶然发现的线索（翻垃圾桶、偷听对话）可用，但每章限用一次，且后续必须有逻辑推进。

### 1.3 动机双重印证原则

- **规则**：凶手/主谋的作案动机，必须在揭秘前通过至少2处独立线索加以铺垫。
- **"独立"的定义**：两条线索来源不同（例：一条来自人证、一条来自物证），且单独看来只是背景信息。
- **目的**：让读者事后回看时发现"原来早有暗示"，实现公平游戏原则。
- **检查方法**：写完揭秘章节后，列出动机的两条铺垫线索，确认均已在文中出现。

### 1.4 公平游戏原则（Fair Play）

- **规则**：读者拥有与主角相同的信息基础，在主角揭秘之前，读者在理论上应能自行推理出答案。
- **操作**：
  - 所有关键物证须在主角"注意到"时，给予读者同等的描述。
  - 主角内心推理可以模糊（增加悬念），但不得有意误导读者（禁止"我知道凶手是谁，但先不说"的叙事技巧，此为推理禁忌）。
  - 凶手不得是直到案件揭秘才第一次出场的人物；凶手须在揭秘前至少出场2次并有实质互动。

## 二、技术水平设定

### 2.1 科技环境

| 技术领域 | 可用程度 | 备注 |
|----------|----------|------|
| 手机/互联网 | 普及 | 可用于追踪、通讯、查资料，但有监控反制手段 |
| 监控摄像头 | 广泛但有盲区 | 老城区、废弃仓库、私人空间为盲区 |
| DNA检测 | 存在但慢且贵 | 警方有权限，主角需关系或费用才能使用 |
| 数字取证 | 初级阶段 | 手机数据可取，但需专业人员；加密通讯难破解 |
| 人脸识别 | 部分地区有 | 主要覆盖公共交通和主干道，老城区缺失 |
| 法医技术 | 中等水平 | 毒物检测准确，但需时间；微物证分析依赖经验 |

### 2.2 信息不对称规则

- 主角获取官方数据库（如户籍、车牌）须通过非官方渠道，需付出代价（金钱、人情、风险）。
- 警方拥有主角不具备的强制搜查权，这是主角的先天劣势，须在剧情中体现。
- 凶手也受相同的技术约束——凶手的完美犯罪方案不得依赖超出设定技术水平的手段。

## 三、叙事规则

### 3.1 章节结构规则

- **单章独立案件**：除第1章（全书引言/主角介绍）外，每个独立案件须在有限章节内形成完整弧线（起案→调查→揭秘→余波）。
- **建议章节分配**（单案）：
  - 轻量案（2章）：委托+揭秘
  - 标准案（3章）：委托→调查/误区→揭秘
  - 重量案（4-5章）：委托→调查→错误推断→反转揭秘→余波
- **过渡章节**：主弧连接章（非独立案件章节）每5章最多1章，用于推进大弧或主角个人线。

### 3.2 主弧线索节奏

- **规则**：主弧（背后阴谋）线索每3-5章必须出现一次，不得超过5章的空白。
- **形式**：
  - 强线索：明显与主弧相关，读者可察觉（ch1-10慎用）
  - 弱线索：伪装成无关细节，读者可能忽视（ch1-10主要使用此类）
  - 人物线索：通过某个配角的异常行为暗示其与幕后的关联
- **节奏设计参考**：
  ```
  Ch1（引入）→ Ch3（弱线索①）→ Ch5（弱线索②）→ Ch7（弱线索③）→
  Ch10（小汇总，主角隐约感觉不对）→ Ch13（强线索①）→...
  ```

### 3.3 主角可犯错原则

- **规则**：主角不是全知全能的，必须在系列中犯至少以下两类错误：
  - **推理失误**：得出错误结论，并因此采取了错误行动，有实际后果。
  - **道德困境失误**：在两难选择中选了"错"的一方，造成不可逆的代价。
- **目的**：保持人物真实感，增加读者代入感和悬念。
- **禁止**：主角犯错后轻易"被原谅"而无实质代价；每章都犯错（过度化）。

### 3.4 暴力描写尺度

- **原则**：暴力服务于叙事，不为猎奇而存在。
- **允许**：
  - 案件现场描写：精确但克制，重点在线索而非残忍细节
  - 主角受伤：真实描写痛感和恢复周期，不允许"神奇自愈"
  - 打斗场景：短促、混乱、有代价，避免好莱坞式英雄无敌
- **禁止**：
  - 对受害者尤其是弱势群体（儿童、老人）的详细施害过程描写
  - 以"酷"为目的的暴力美化
  - 性暴力的细节化描写（可作为案情事实陈述，但不得渲染）
- **分级参考**：本系列定级为"成人向现实主义推理"，等同于国内出版可接受上限。

## 四、人物一致性规则

### 4.1 配角行为逻辑

- 所有有名字的配角必须有一个明确的"行为动机公式"，其在每次出场时的行为须符合该公式。
- 动机公式示例：【老警察王队：保全仕途 > 维护正义，但有底线（不杀无辜）】
- 若配角行为需要改变，须有铺垫（受到某事件冲击、信息量改变等）。

### 4.2 凶手设计规则

- 凶手必须具有以下三要素，且均须在揭秘前有所铺垫：
  1. **动机**（为什么要犯案）
  2. **能力**（有没有条件实施犯案）
  3. **机会**（在案发时间地点能否出现）
- 任何一项缺失，揭秘均不成立。

### 4.3 禁用叙事技巧

| 技巧 | 原因 |
|------|------|
| 双胞胎/冒充者（未提前铺垫） | 违反公平游戏 |
| 主角因本能/直觉猜中凶手 | 破坏推理逻辑 |
| 关键证物凭空出现 | 违反信息渠道规则 |
| 结尾用凶手自供代替推理 | 规避了逻辑验证 |
| 所有嫌疑人都是共谋 | 稀释个体动机，削弱戏剧张力 |

## 五、系列连贯性规则

- **人物记忆**：主角和重要配角对上一案件的经历须有记忆残留，体现在态度、习惯或对话细节中。
- **世界累积变化**：发生过的重大事件（有人死亡、主角受重伤、线人离开）须在后续章节中有体现，不得重置。
- **证据链保存**：每案揭秘后，须简述案件在现实中的法律/社会结果（是否起诉、是否定罪），保持世界真实感。
