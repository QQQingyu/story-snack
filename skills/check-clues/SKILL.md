# 查看线索

展示完整的线索追踪状态。

## 执行

1. 读取 `clues/tracker.json`

2. 按类型分组展示：

### 活跃真线索
| ID | 埋设章节 | 描述 | 呈现方式 |
|---|---|---|---|

### 活跃红鲱鱼
| ID | 埋设章节 | 描述 | 误导方向 |
|---|---|---|---|

### 暗线伏笔
| ID | 埋设章节 | 描述 | 关联阶段 |
|---|---|---|---|

### 已解决/已揭穿
| ID | 类型 | 埋设章节 | 解决章节 | 描述 |
|---|---|---|---|---|

3. 读取 `clues/arc-progress.json` 展示暗线进度：
   - 当前阶段
   - 已埋伏笔 vs 计划伏笔
   - 已揭示 vs 计划揭示
