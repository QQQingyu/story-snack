# 检查状态

快速展示当前写作进度。

## 执行

1. 读取 `clues/arc-progress.json`：
   - 当前章节号
   - 主弧阶段（铺垫/升级/高潮）
   - 已埋主弧伏笔数量
   - 张力等级

2. 读取 `clues/tracker.json`：
   - 活跃伏笔数量
   - 活跃误导数量
   - 已回收叙事线数量

3. 读取最近 3 章摘要（`manuscript/summaries/` 下最新 3 个文件）

4. 以简洁表格展示：

```
Story Snack 进度

章节：第 N 章已完成
主弧：{阶段名}（张力 {N}/10）
叙事线：{N} 条活跃 | {N} 条已回收
伏笔：{N} 个已埋设

--- 近 3 章回顾 ---
第 X 章：{摘要}
第 Y 章：{摘要}
第 Z 章：{摘要}
```
