# 查看叙事线

展示完整的叙事线追踪状态。

## 执行

1. 读取 `clues/tracker.json`

2. 按类型分组展示：

### 活跃伏笔
| ID | 埋设章节 | 描述 | 呈现方式 |
|---|---|---|---|

### 活跃误导
| ID | 埋设章节 | 描述 | 误导方向 |
|---|---|---|---|

### 主弧线索
| ID | 埋设章节 | 描述 | 关联阶段 |
|---|---|---|---|

### 已回收/已揭穿
| ID | 类型 | 埋设章节 | 回收章节 | 描述 |
|---|---|---|---|---|

3. 读取 `clues/arc-progress.json` 展示主弧进度：
   - 当前阶段
   - 已埋伏笔 vs 计划伏笔
   - 已揭示 vs 计划揭示
