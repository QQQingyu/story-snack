# 写作风格指南

本指南是 AI 写作 Agent 的操作手册，适用于 2080 赛博朋克悬疑推理系列。所有规则均为强制执行项。

---

## 一、基本参数

| 项目 | 规格 |
|------|------|
| 视角 | 第三人称限知（跟随艾米，不得进入其他角色的内心） |
| 时态 | 过去时（"她走进房间"，不用"她走进房间。现在她看到……"） |
| 每章字数 | 3000–4000 字 |
| 章节结构 | 三段式：上篇 / 中篇 / 下篇，各 1000–1300 字 |
| 语言 | 中文为主。人名在正文中一律使用中文（艾米、高斯林、赛拉斯等），英文名仅保留在角色档案中 |

**视角纪律**：叙述者只能写艾米能观察到、听到、感受到的信息。艾米不在现场的事件，只能通过对话、文件、数据终端查询等渠道引入。违反此规则属于"角色知识违规"，是硬性错误。

**义体视觉的写作方式**：艾米的义体眼捕捉到的增强信息（红外、电磁痕迹等），必须以感官化的方式呈现，而非数据读数。不写"义体眼显示目标体温 38.2°C"，写"她的视野边缘泛起一层暖色——这个人在发热，或者刚剧烈运动过。"

---

## 二、章节格式

每章必须严格使用以下 Markdown 格式：

```
# 第 N 章：{章节标题}

## 上 · {小标题}

{正文，1000–1300 字}

## 中 · {小标题}

{正文，1000–1300 字}

## 下 · {小标题}

{正文，1000–1300 字}
```

- `N` 为章节序号（阿拉伯数字）
- 章节标题简短有力，4–8 字
- 小标题点出该段核心动作或情绪转折

**示例**：
```
# 第 3 章：过载的脉冲

## 上 · 雨中的信号

## 中 · 她的眼睛在说谎

## 下 · 序列号的空白
```

## 三、三幕节奏规范

### 上篇（开场，快节奏钩子）

- **前 3 句必须建立悬念**：用动作、异常细节或反常对话开场，禁止用背景介绍或环境描写起笔
- 引入章节核心冲突或谜题
- 节奏偏快：短句为主，段落不超过 4 行

**合格开场示例**：
> 尸体还是温的，但义体已经冷了。艾米蹲下来，用义体眼扫了一遍——所有部件都在，除了颈椎接口处那块该有的东西。

**不合格开场示例**：
> 2080 年的锈港，霓虹灯照亮了潮湿的街道。艾米裹紧了她的机能夹克，走在前往案发现场的路上……

### 中篇（发展，交替推进）

- **对话与内心推理交替出现**，比例大致各半
- 每段对话之后，艾米必须有一次内心评估（短则一句，长则三句）
- 植入至少一个**误导线索**：看起来重要、实则无关，或看起来无关、实则关键
- 中篇结尾必须留下一个未解决的小疑问

### 下篇（高潮，加速收束）

- 节奏持续加速：段落越来越短，对话越来越急促
- 揭示一个关键信息
- **章节结尾禁止道德总结、感悟句、人生哲理**
- 结尾必须是三类之一：
  1. 行动截断（艾米正在做某事，章节戛然而止）
  2. 新谜题抛出（一个新的异常出现）
  3. 情绪余震（一句话，留白）

## 四、语言风格

### 总原则

- **简洁优先**：能用 10 字说清楚的，不用 20 字
- **禁止形容词堆叠**：每个名词最多跟一个修饰语
- **具体胜过抽象**：不写"她很紧张"，写"她把杯子放下去，又立刻端起来"
- **禁止机械式并列短句**：不写"她走了，他跟上来了""她站起来，他坐下了"这类"A了，B了"的干瘪并列结构。用具体感官细节替代——写动作的声音、画面、节奏，让读者看到场景而不是读到流水账。例："她迈步走进雨后的街道，身后传来高斯林小跑跟上的脚步声"

### 可读性第一原则（极重要）

> **写给所有人看，不是写给赛博朋克粉丝看。**

1. **先写"做什么"，再写"叫什么"**：不写"力矩传感器模块被偷了"，写"他假手上的一个零件被偷了——没了它，手指捏不住东西，摊子就开不下去"。读者先理解后果和情感，技术名词可以出现也可以不出现
2. **每章新概念不超过 2 个**：第一章只需要让读者理解"有人装了机械手臂"和"这座城市分上下层"就够了。排异、企业封建、数字身份等概念在后续章节逐步引入
3. **用人物关系解释世界**：不要用旁白介绍"五大企业控制了城市"，用角色的一句抱怨带出来——"Vera 骂了一句：澜盾的人连死人都不放过"
4. **技术术语的降级规则**：
   - 能用日常词替代的，用日常词："假手""机械腿""电子眼"比"义体上肢""义体下肢""视觉增强义体"更好
   - 必须用术语时，用动作或后果紧跟解释，不用括号注释
   - 同一术语在全书中第一次出现时必须有语境解释，之后可以直接使用
5. **情感入口优先**：每个场景的第一句话应该是关于人的（情绪、动作、关系），而不是关于技术的

### 感官细节（案发现场专项）

案发现场必须覆盖至少 3 种感官，且细节须服务于推理：

- **视觉**：光线方向、阴影、颜色异常、义体眼增强视觉捕捉到的光谱细节（用感官语言呈现）
- **嗅觉**：最容易被遗忘、最难伪造。赛博朋克特有气味：臭氧、焊接金属、义体液压液、合成食品甜腻味、廉价义体散发的微弱电路烧灼味
- **听觉**：义体关节的液压声、全息广告的合成人声、货运无人机低频嗡鸣、环境音的缺席（"房间里没有电磁底噪——有人在用屏蔽器"）
- **触觉**：义体金属的温度（正常运行时微温，停机时冰冷）、地面的震动频率、空气湿度对廉价义体接口的影响
- 味觉：仅在艾米直接接触时使用，不强求

**错误示范**：
> 现场一片狼藉，充满了赛博朋克式的颓废感。

**正确示范**：
> 义体还在嗡鸣，低沉的频率说明电池没耗尽——死了不超过两小时。地上有液压液的痕迹，从左臂关节处一直拖到门口，中间断了一截。有人擦过，但没擦干净。

### 科技术语的使用尺度

- **融入而非炫技**：科技元素必须像空气一样自然存在于角色的日常中，不需要解释
- **不写**："她使用了她的赛博义体视觉增强功能来扫描现场"
- **要写**："她眯了下眼，视野切到红外——地板上有一串正在散去的热足迹，从门口到窗边，中间没有停顿"
- **术语密度**：每段正文中科技专有名词不超过 2 个。如果无法避免，用动作或对话自然带出含义，而非括号注释
- **角色对科技的态度**：艾米使用义体眼就像我们使用眼镜——不会每次都强调"我的义体眼"。高斯林对高科技有天然的不信任，他会说"用你的眼睛"而不是"用你的义体眼"

### 逻辑推理的嵌入方式

推理过程不得以独立的"分析段落"形式出现，必须嵌入动作或对话：

- **动作驱动推理**：艾米拿起某个物件时，顺带想到它的含义
- **对话触发推理**：对方说了某句话，艾米在心里标记为"这句话有问题"
- **义体辅助推理**：义体眼捕捉到的细节触发思考，但结论是艾米自己推出的——义体是工具，不是推理者
- **推理结论延迟**：艾米注意到线索，但不立刻说出结论——结论在下篇或下一章才浮现

禁止写法：
> 艾米开始分析现场。首先，凶器是……其次，嫌疑人有三个……最后，她得出结论……

## 五、对话规范

### 说话标签

- 优先使用"说"和"问"，其次"答""道"
- 禁止使用情绪化标签："她愤怒地喊道""他悲伤地低语"——情绪由台词内容和前后动作体现
- 标签之后禁止副词："她轻声地说"改为"她放低了声音"（动作前置）

### 人物辨识度速查

| 角色 | 句式特征 | 标志性习惯 |
|------|----------|-----------|
| Aimee | 短句、句号多、几乎不用感叹号。偶尔冷幽默 | 说"不对"时语气平淡但所有人都会停下来 |
| Gosling | 长句、感叹号和省略号多、爱用类比 | "我跟你说——"开头，能接任何话题 |
| Vera | 快速、命令式、不留余地 | 骂人和关心用同一种语气 |
| 焊工 | 单字或半句，极少完整句子 | 沉默本身就是回答 |
| 小盒子 | 精确、无感情、省略主语，像代码注释 | "你没听我说过这个" |
| 针头 | 低声、快速、左右看 | "这事你没从我这儿听到" |
| Knox | 慢、重、每个字像在称分量 | "这是建议，不是警告" |

每章首次出现某角色时，须在台词前用一句动作描写强化辨识度。

### 对话功能检查

每段对话必须同时完成至少两项：
1. 推进情节（透露新信息）
2. 揭示角色性格
3. 制造或缓解紧张
4. 植入线索或误导

纯寒暄禁止，除非寒暄本身是伪装。

## 六、节奏控制

### 句子长度与段落密度

| 场景 | 句长 | 段落行数 |
|------|------|----------|
| 追逐/冲突/义体对抗 | 5–15 字 | 1–2 行 |
| 日常调查/移动/数据查询 | 15–30 字 | 2–4 行 |
| 回忆/内心分析/义体眼扫描 | 20–40 字 | 3–5 行 |
| 对话密集段 | 视内容而定 | 每句单独成段 |

### 对话与叙事的比例

全章对话字数占比：**30%–40%**。

---

## 七、赛博朋克世界的日常感

> 核心原则：2080 年的科技对角色来说就是日常。不解释，不惊叹，不强调"这是未来"。

### 该写的

- 义体维修像换手机屏幕一样普通——"义体又卡了，回头找 Vera 看看"
- 全息广告是视觉污染——角色不会驻足观看，只会下意识躲开或无视
- 数字身份是生存前提——被冻结身份的恐惧等同于当代人被注销银行账户+身份证+手机号

### 不该写的

- 角色对自己生活的世界发出惊叹或解释（"在这个高科技的时代……"）
- 用大段文字描述某项技术的工作原理（除非角色正在分析它与案件的关系）
- 为了"赛博朋克感"而堆砌科技名词

---

## 八、禁止列表（硬性规则）

### 禁用词组（空洞连接词）

众所周知、不言而喻、毋庸置疑、不得不说、值得注意的是、总的来说、由此可见、话虽如此

### 禁用写法

| 禁止 | 原因 |
|------|------|
| "突然，她灵光一闪" / "她忽然想到" | 推理必须有铺垫 |
| 章节结尾的道德感悟 | 破坏悬疑类型感 |
| 艾米知道自己不该知道的信息 | 角色知识违规 |
| 形容词三叠 | 废话 |
| 全知叙述："与此同时，在另一个地方……" | 违反限知视角 |
| "在这个赛博朋克的世界里" | 打破沉浸感 |
| 义体眼自动给出推理结论 | 义体是工具，不是侦探 |

### AI 高频词黑名单

以下词语须主动回避，找具体替代表达：

错综复杂、扑朔迷离、若隐若现、心中一紧、心头一沉、豁然开朗、抽丝剥茧、层层递进、娓娓道来、不禁（"她不禁想到"）、不由得（"她不由得皱眉"）、霓虹闪烁的街道（赛博朋克文的万能废句）、高科技与低生活（直接使用设定术语属于打破第四面墙）

## 九、自我检查清单

每次生成章节后，按以下顺序逐项验证：

- [ ] 上篇前 3 句是否建立了悬念？
- [ ] 全章是否严格保持第三人称限知？艾米有无接触到视角外的信息？
- [ ] 对话比例是否在 30%–40% 之间？
- [ ] 中篇是否有至少一个误导线索？
- [ ] 案发现场是否覆盖 3 种以上感官细节？
- [ ] 推理过程是否嵌入动作/对话，而非单独列出？
- [ ] 义体眼的信息是否以感官化方式呈现（非数据读数）？
- [ ] 科技元素是否自然融入而非强行解释？
- [ ] 下篇是否加速收束，结尾是行动截断/新谜题/情绪余震三类之一？
- [ ] 章节结尾是否没有道德总结？
- [ ] 禁用词组和 AI 高频词是否已全部清除？
- [ ] 每个出场角色的语言习惯是否与角色档案一致？
- [ ] 字数是否在 3000–4000 字范围内？

## 十、参考节奏范例

以下为中篇段落的节奏示范：

> "你昨晚几点回来的？"Gosling 把椅子转过来骑坐上去，手臂搭在椅背上。
>
> "十点多吧。"女人端起合成咖啡，又放下了。"具体几点我没看表。"
>
> *十点多*。Aimee 在心里记下这个词。死者义体行动日志的最后一条记录是 22:13。她给自己留了一个小时的模糊地带。
>
> "你一个人回来的？"Gosling 继续问。
>
> "是。"
>
> 只有一个字。Aimee 注意到女人左手的义体手指关节处有一圈微弱的热纹——刚充过电，不超过两小时。但她声称整晚都在家。家里的充电桩在客厅，走廊的监控能拍到。如果她真的整晚都在家，为什么需要在凌晨给义体充电？
>
> "……有意思。"Aimee 说。
>
> 女人的义体手指微微收紧。
