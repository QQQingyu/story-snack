## 三、章节格式

每章必须严格使用以下 Markdown 格式：

```
# 第 N 章：{章节标题}

## 上 · {小标题}

{正文，1000-1300 字}

## 中 · {小标题}

{正文，1000-1300 字}

## 下 · {小标题}

{正文，1000-1300 字}
```

- `N` 为章节序号（阿拉伯数字）
- 章节标题简短有力，4-8 字
- 小标题点出该段核心动作或情绪转折

## 四、三幕节奏规范

### 上篇（开场，快节奏钩子）

- **前 3 句必须建立悬念或冲突**：用动作、异常细节或反常对话开场，禁止用背景介绍或环境描写起笔
- 引入章节核心冲突
- 节奏偏快：短句为主，段落不超过 4 行

### 中篇（发展，交替推进）

- **对话与内心活动交替出现**，比例大致各半
- 每段对话之后，视角人物应有一次内心评估（短则一句，长则三句）
- 中篇结尾必须留下一个未解决的疑问或悬念

### 下篇（高潮，加速收束）

- 节奏持续加速：段落越来越短，对话越来越急促
- 揭示一个关键信息或产生重要转折
- **章节结尾禁止道德总结、感悟句、人生哲理**
- 结尾必须是三类之一：
  1. 行动截断（角色正在做某事，章节戛然而止）
  2. 新悬念抛出（一个新的异常或未知出现）
  3. 情绪余震（一句话，留白）

## 五、语言风格

### 总原则

- **简洁优先**：能用 10 字说清楚的，不用 20 字
- **禁止形容词堆叠**：每个名词最多跟一个修饰语
- **具体胜过抽象**：不写"她很紧张"，写"她把杯子放下去，又立刻端起来"
- **禁止机械式并列短句**：不写"她走了，他跟上来了"这类干瘪并列结构。用具体感官细节替代

### 可读性第一原则

> **写给所有人看，不是写给类型文学粉丝看。**

1. **先写"做什么"，再写"叫什么"** — 读者先理解后果和情感，专业术语可出现可不出现
2. **每章新概念不超过 2 个** — 世界观元素逐步引入，不一次性倾倒
3. **用人物关系解释世界** — 不要用旁白介绍设定，用角色的抱怨、回忆、对话自然带出
4. **情感入口优先** — 每个场景的第一句话应该是关于人的（情绪、动作、关系），而不是关于设定的

### 感官细节

重要场景必须覆盖至少 3 种感官，且细节须服务于叙事：

- **视觉**：光线方向、阴影、颜色异常
- **听觉**：环境音、对话节奏、沉默
- **嗅觉**：最容易被遗忘的感官，往往最有记忆点
- **触觉**：温度、质感、震动
- 味觉：仅在角色直接接触时使用，不强求

## 六、对话规范

### 说话标签

- 优先使用"说"和"问"，其次"答""道"
- 禁止使用情绪化标签："她愤怒地喊道""他悲伤地低语"——情绪由台词内容和前后动作体现
- 标签之后禁止副词："她轻声地说"改为"她放低了声音"（动作前置）

### 人物辨识度

每章首次出现某角色时，须在台词前用一句动作描写强化辨识度。

### 对话功能检查

每段对话必须同时完成至少两项：
1. 推进情节（透露新信息）
2. 揭示角色性格
3. 制造或缓解紧张
4. 植入伏笔或误导

纯寒暄禁止，除非寒暄本身是伪装。

## 七、节奏控制

### 句子长度与段落密度

| 场景 | 句长 | 段落行数 |
|------|------|----------|
| 追逐/冲突/高潮 | 5-15 字 | 1-2 行 |
| 日常推进/移动/调查 | 15-30 字 | 2-4 行 |
| 回忆/内心活动/分析 | 20-40 字 | 3-5 行 |
| 对话密集段 | 视内容而定 | 每句单独成段 |

### 对话与叙事的比例

全章对话字数占比：**30%-40%**。

## 八、禁止列表（硬性规则）

### 禁用词组（空洞连接词）

众所周知、不言而喻、毋庸置疑、不得不说、值得注意的是、总的来说、由此可见、话虽如此

### 禁用写法

| 禁止 | 原因 |
|------|------|
| "突然，她灵光一闪" / "她忽然想到" | 转折必须有铺垫 |
| 章节结尾的道德感悟 | 破坏类型感 |
| 角色知道自己不该知道的信息 | 角色知识违规 |
| 形容词三叠 | 废话 |
| 全知叙述（限知视角下）："与此同时，在另一个地方……" | 违反限知视角 |

## 九、自我检查清单

每次生成章节后，按以下顺序逐项验证：

- [ ] 上篇前 3 句是否建立了悬念或冲突？
- [ ] 全章是否严格保持设定的叙事视角？
- [ ] 对话比例是否在 30%-40% 之间？
- [ ] 重要场景是否覆盖 3 种以上感官细节？
- [ ] 下篇是否加速收束，结尾是行动截断/新悬念/情绪余震三类之一？
- [ ] 章节结尾是否没有道德总结？
- [ ] 禁用词组和 AI 高频词是否已全部清除？
- [ ] 每个出场角色的语言习惯是否与角色档案一致？
- [ ] 字数是否在 3000-4000 字范围内？
